## 一、整日大循环

- [~] 将现有“清晨人 → 白天鸡 → 黄昏鸡呼叫人 → 人撒食赶鸡 → 夜间结算”改为：
  - 清晨由人观察、互动、购买和准备。
  - 白天完整交给鸡，保持连续探索心流。
  - 黄昏仍由鸡行动，玩家可安全回窝，也可被夜间食物吸引而留下冒险。
  - 人不在白天作为自主 NPC 出现，也不在黄昏切回来赶鸡。
  - 次日清晨由蛋、粪便、花、羽毛、坑和动物痕迹呈现昨天的结果。
- [~] 删除鸡在黄昏连续叫人、切换成人、撒无限引路食物、开关鸡舍门的固定劳动流程。
- [~] 普通碎谷不再承担黄昏引路核心玩法；特别瓜子只保留为清晨人与鸡之间的特别喂食。
- [ ] 黄昏进入夜晚时不弹任务窗口：鸡舍暖光表示安全结束，远处虫声和动静自然邀请玩家继续冒险。
- [~] 第 14 天不再围绕旧版“暴风中由人赶鸡关门”收束；应使用玩家两周实际留下的院子、领地、道具、关系和事件记忆完成回顾与升华，具体流程见“第 14 天回顾、升华与自由模式”。

## 二、蛋、预算与动态购买

- [~] 将当前“每颗蛋固定换一份木料、次日送达”的经济改为蛋品质决定院子预算。
- [ ] 每天清晨找到前一天的蛋后，根据内部品质获得预算：
  - 差蛋：2。
  - 普通蛋：3。
  - 好蛋：4。
  - 极好蛋：5。
- [ ] 蛋品质由前一天体验形成：
  - 基础值始终保底为 2。
  - 达到舒服的饱食状态可提高品质。
  - 吃到野生虫或形成丰富饮食可继续提高。
  - 在干燥、舒适的地方休息可继续提高。
  - 淋雨最多让次日蛋降低一级，不产生连续惩罚。
- [ ] 特殊蛋保留自己的外观和记忆意义，同时按其内部普通品质提供预算，不要求出售或放弃收藏。
- [ ] 所有可购买道具从一开始都能看见，不按日期解锁；前中后期仅由价格、蛋的质量和玩家选择自然形成。
- [ ] 好蛋让玩家更快购买道具，差蛋只减慢进度，不能形成“蛋差 → 买不起改善 → 蛋更差”的死循环。
- [ ] 第一场大雨前的最低收入必须足以购买水盆和加固鸡窝：
  - 第一场大雨计划在第 7 天，并于第 5～6 天预报。
  - 第 7 天清晨最差路线累计至少 14 预算。
  - 水盆 3、加固鸡窝 5；即使玩家此前购买灯泡和优质饲料，也存在足够容错。
- [~] 当前蛋相册与多种蛋类型可以复用，但需要增加音乐蛋、冒险蛋、夜食蛋等“记录昨天事件”的特殊蛋。

## 三、鸡的身体与自然体验

- [ ] 增加真正围绕夏天的体热循环：
  - 冲刺、弹跳、追虫和烈日使鸡变热。
  - 树荫可以缓慢降温。
  - 喝水可以快速降温。
  - 凉坑和安静休息可以持续恢复。
  - 不掉血、不突然禁止移动，通过张翅、喘气、步态和声音表现。
- [~] 当前体力、饱食、冲刺能量和水增益需要收束为玩家能从鸡的动作理解的身体状态，避免依赖精确 HUD 数字。
- [ ] 饥饿不作为生存惩罚；基础饲料永远存在，虫子和优质饲料主要决定饱食舒适度与蛋品质。
- [ ] 梳理羽毛、沙浴、翻滚和抖尘成为本身有满足感的动作，不设置必须填满的“羽毛清洁条”。
- [ ] 鸡在雨中淋湿后变成明显的“落汤鸡”，动作和羽毛外观改变；不会生病或永久受罚。
- [ ] 次日清晨允许人用毛巾抱着擦干鸡，作为雨天失败后的温暖释放。

## 四、探索、熟悉度与长期领地

- [ ] 每片区域拥有隐藏熟悉度，不使用战争迷雾和完成度进度条。
- [ ] 鸡第一次进入陌生区域时放慢、伸长脖子观察并对声音敏感；熟悉后记住虫点、水、树荫、安全路线和夜间逃生路径。
- [ ] 野生虫的可获得数量随实际探索和区域熟悉而增加，而不是仅按固定日期解锁。
- [~] 当前会在一两天内消失的简单坑需要改为可长期保留、反复加深的领地痕迹。
- [ ] 鸡最多形成两三个长期领地，类型由实际行为自然形成，不在菜单中指定：
  - 凉坑：阴凉或略湿，反复刨深后提高降温与休息效果。
  - 沙浴地：干燥松软，用于翻滚、抖尘和梳羽；下雨或喷水后会暂时变湿。
  - 观察／安全休息地：树根、灌木或遮挡附近，用于观察院门、躲避动物和黄昏休息。
- [ ] 新领地不会立刻删除旧领地；需要多次使用后才逐渐替代最少使用的地点。
- [~] 当前松土区、遮阴棚、低栖木及其休息动画可以作为领地行为基础，但不能代替动态形成的长期领地。

## 五、夜间食物与黄鼠狼

- [~] 黄鼠狼从当前第 8 天后出现、可由人用手提灯驱赶的事件，改为前中期持续存在的核心压力。
- [ ] 前中期没有能攻击、困住或直接赶走黄鼠狼的工具；玩家只能提前察觉、判断风险、利用熟悉路线并逃回鸡舍。
- [ ] 夜间长期诱惑“月光虫”：
  - 只在夜晚院子边缘的泥土中活动。
  - 通过泥土鼓动和细小声音提示，不显示任务箭头。
  - 鸡需要原地连续刨几秒才能挖出。
  - 刨土声会增加黄鼠狼发现鸡的机会。
  - 吃到后出现强烈满足动作，并可能形成夜食蛋。
- [ ] 门外灯泡只照亮房屋附近的小范围，让鸡看见少量夜虫和危险动静；不直接驱赶黄鼠狼。
- [ ] 每个夜晚第一次靠近门外灯泡时，立即降低 5～6 点夜压；同一夜只能触发一次。触发后灯仍提供照明，但不能反复进出刷减压。
- [~] 黄鼠狼行为需要改为可读的三段压力：
  - 草丛晃动、贴地影子或反光眼睛。
  - 鸡压低身体并朝声音方向观察。
  - 确认位置后追逐。
- [ ] 每次夜行最多发生一次完整追逐，避免连续压力。
- [~] 被黄鼠狼抓到不能杀死鸡或清除进度：
  - 爆出羽毛并丢失尚未吃下的夜虫。
  - 鸡慌张逃回鸡舍，夜晚结束。
  - 次日人能看到羽毛和受惊反应，并通过抱鸡、摸鸡完成情绪释放。
- [ ] 鸡进入鸡舍可以安全结束夜晚；获得后背大灯后可以在长期领地或树下户外过夜。
- [~] 删除旧版人类手提灯照射、人体阻挡黄鼠狼和黄昏关门作为主要解决方法。

## 六、粪便、扫地与花

- [ ] 鸡视角每天产生五次粪便：
  - 时间随机。
  - 相邻两次必须有明显间隔。
  - 不显示排泄倒计时、提示图标或排泄数值。
  - 只用短暂自然动作表现，粪便较小并留在地面。
- [ ] 人踩到粪便时：
  - 双手高举并进入慌张走路动作。
  - 不夺走移动控制，不扣预算或关系。
  - 接下来几步留下逐渐变淡的粪便脚印。
  - 被踩中的粪便变成污迹，数秒后与脚印一起消失。
- [ ] 清晨可以拿扫把将多处粪便扫成一个小肥料堆，不要求逐个装入背包。
- [ ] 肥料只能放在院内泥地，不能放在房屋内部或道路；放置时显示小范围有效预览。
- [ ] 放置后的第二天，以该点为圆心长出五至八朵有明显间距的小花。
- [ ] 新花只填补已有花朵之间的空隙，不能形成密集花毯；旧花建议在五至七天后逐渐稀疏。
- [ ] 扫粪和种花始终是可选活动，不成为结束清晨的条件，也不提供额外数值奖励。

## 七、人视角商店与可获取道具

以下道具均不按日期开放；价格只是形成自然的前中后期节奏。

### 低价、偏现实

- [ ] 门外灯泡（2）：安装一次，夜晚照亮房屋附近少量夜虫和环境动静，不驱赶黄鼠狼。每夜第一次靠近时减少 5～6 点夜压，且只能生效一次；鸡会放松身体、轻抖羽毛，让减压通过动作被看见。
- [ ] 水盆（3）：人提壶加水，鸡饮水后快速降温；装满后可使用多天。
- [ ] 优质饲料桶（4）：清晨舀一勺，使鸡以较舒服的基础饱食开始一天，但不能替代野外抓虫获得最高品质蛋。
- [~] 遮阴布（4）：基于当前遮阴棚重做，可搭在已有领地上并让该地点逐渐成为凉坑。
- [ ] 加固鸡窝（5）：一次性加固屋顶，雨天不漏水；未购买时鸡会淋湿，但不会生病。

### 中价、改变习惯

- [~] 有顶沙浴箱（6）：基于当前松土区重做，倒入细沙后在雨天仍可使用；需由鸡多次使用才成为领地。
- [~] 木箱瞭望台（6）：基于当前低栖木扩展，让鸡观察远处虫点，并可能发现院外、触发逃跑事件。
- [ ] 旋转喷水器（8）：转动阀门后大范围降温并产生玩水动作；会暂时弄湿附近沙浴地。
- [ ] 音箱（10）：人拧动大旋钮后切入鸡视角跳舞；出现舞池灯光、动物反应和次日音乐蛋。

### 高价、规则变化

- [ ] 弹跳垫（11）：鸡通过连续按键弹跳，捕捉空中虫；会快速升温，使水盆、树荫和喷水器继续有价值。
- [ ] 夜虫幕布（12）：清晨展开或卷起；夜晚聚集大量飞虫，也让鸡更容易暴露给黄鼠狼。
- [ ] 后背大灯（18）：
  - 人清晨决定安装或取下，鸡当天不能中途切换。
  - 使鸡全天普通移动、起步和转弯变慢。
  - 黄鼠狼准备扑击时自动爆亮并将其吓走。
  - 爆亮也会惊走附近夜虫。
  - 安装后允许鸡在户外领地过夜。
  - 与音箱组合时可形成完整舞台效果。

## 八、音乐、逃跑与其他记忆事件

- [ ] 音乐播放时，鸡从疑惑歪头过渡到跳跃；麻雀跟跳、猫换地方睡，头顶出现克制的舞池灯光。
- [ ] 使用原创、具有荒诞活力的音乐，商业版本不直接使用未授权歌曲录音或歌词。
- [ ] 鸡偶尔从院门逃跑，触发与主场景相同低多边形风格的 2D 横版追逐：
  - 玩家连续点击鼠标追赶。
  - 接近鸡时进入慢动作。
  - 玩家在正确时机点击将鸡抱住。
  - 失败后立即继续追，不扣预算、关系或进度。
  - 成功以安静抱鸡收尾，并可能形成冒险蛋。
- [ ] 逃跑事件必须低频、有前兆，不能成为每日劳动。

## 九、天气、动物与环境组合

- [~] 当前 14 天随机天气日历需要增加首轮节奏保障：第 5～6 天明确预报，第 7 天首次保证大雨。
- [ ] 雨水会改变坑、沙浴地、泥地、鸡舍和鸡的羽毛表现，而不只改变移动或数值。
- [ ] 雨后增加部分虫类活动，形成探索机会。
- [~] 麻雀从当前主要偷食行为调整为院子的观众和喜剧放大器；可以靠近食物，但不能造成明显资源惩罚。
- [~] 野猫从简单访问事件调整为会占据休息地点、挡住熟悉路线或让鸡谨慎绕行的院中角色，不伤害鸡。
- [ ] 道具组合产生少量不预告的场景反应：
  - 音箱＋喷水器：喷水舞会。
  - 音箱＋后背大灯：完整小舞台。
  - 凉坑＋树荫：稳定午睡领地。
  - 夜虫幕布＋无大灯：最高夜食收益与最高黄鼠狼风险。

- [~] 当前关系记忆、摸头、顺背、轻抱和食物偏好可继续复用，但正式玩法不直接显示亲密度数字；数字仅保留在调试面板。

## 十一、第 14 天回顾、升华与自由模式

### 结局要表达的内容

第 14 天不是“全部指标达标”的胜利，也不评价玩家是否把鸡照顾得完美。它要表达：

> 最开始这里只是一块陌生院子；十四天后，水盆、坑、花、羽毛、灯光、脚印和鸡记住的路线，已经共同把它变成了家。

- [ ] 不使用分数、评级、最佳路线、遗漏道具或“完成度百分比”作为结局核心。
- [ ] 未购买全部道具、淋过雨、被黄鼠狼抓到或产过差蛋，都不能导向坏结局。
- [ ] 玩家此前的小失误也可以成为温暖或好笑的回忆，例如落汤鸡、踩粪脚印和追鸡失败。

### 第 14 天当天

- [ ] 第 14 天仍然是完整可玩的普通一天，不在清晨突然弹出结算界面。
- [ ] 清晨找到第 14 枚蛋后，只给出一句克制提示，例如“今天的院子，好像已经很熟悉了。”
- [ ] 白天由鸡自由活动；经过重要地点时，可以出现极短的声音或动作回声，但不夺走控制：
  - 第一个或最深的长期领地。
  - 最常使用的水盆、树荫或沙浴地。
  - 第一片由粪便变成的花。
  - 音箱与第一次跳舞的位置。
  - 曾经逃跑、被抱住或躲过黄鼠狼的路线。
- [ ] 黄昏后仍由玩家决定安全回窝或进行最后一次夜间冒险，结局不强迫一种“正确选择”。

### 个性化回顾

- [ ] 第 14 天结束后，镜头在清晨缓慢经过当前真实院子，而不是播放与玩家无关的预制动画。
- [ ] 系统从实际存档中挑选四至六段最有辨识度的记忆，每类最多一段：
  - 探索：第一次发现野虫、最熟悉区域或形成第一个领地。
  - 照料：第一次喝水、雨中保持干燥，或淋成落汤鸡后被擦干。
  - 感情：第一次手喂特别瓜子、允许摸头、被抱住或主动靠近人。
  - 荒诞：音乐舞会、喷水舞会、弹跳抓虫或后背大灯照退黄鼠狼。
  - 危险：第一次夜行、吃到月光虫、逃过或被黄鼠狼抓到。
  - 院子变化：花丛、长期坑、屋顶、灯泡或最常使用的设施。
- [ ] 回忆使用短暂半透明动作剪影、熟悉音效和现在的场景叠合，不切成冗长剧情片。
- [ ] 没有发生过的事件绝不伪造；玩家应当能认出“这确实是我经历过的那一天”。
- [ ] 旁白只承担串联作用，保持简短，不逐项解释机制。建议核心语句：
  - “最开始，它只敢在树影边停一停。”
  - “后来，它记住了水、泥土，也记住了你的脚步声。”
  - “你们没有把每一天都过得完美。”
  - “但这个院子，已经到处都是一起生活过的样子。”

### 最后的玩家操作

- [ ] 回顾结束后返回第 15 天清晨的真实院子，由玩家控制人走向鸡，而不是点击“领取结局”。
- [ ] 玩家按下已经熟悉的互动键并蹲下；鸡主动走完最后一小段距离：
  - 关系较浅时，鸡在一步之外安静卧下。
  - 关系较深时，鸡贴近腿边或主动钻进怀里。
  - 所有关系路线都温暖，不设置冷漠或惩罚结局。
- [ ] 镜头缓慢拉远，完整展示玩家自己的院子。最后一句为：“这里已经是它认得的家。”
- [ ] 标题出现后不清空或回滚存档，直接进入第 15 天自由模式。

### 自由模式

- [ ] 保留预算、已购道具、花、领地深度、关系、蛋相册、探索熟悉度和动物记忆。
- [ ] 天气、蛋、夜虫、黄鼠狼和特殊事件继续循环，未买到的道具仍可购买。
- [ ] 不再增加新的强制教程或难度层级，让玩家继续享受已经形成的小院生活。
- [ ] 回顾内容存入蛋相册中的“十四天”页面，玩家可以重看，但不会重复打断自由模式。

## 十二、统一美术风格

### 风格定位

后续全部角色、场景、道具、动画和特效统一采用：

> 轻松、惬意、带克制荒诞感的低多边形 3D 小院；画面像一个可亲手摆弄的玩具舞台，幽默主要依靠清楚的轮廓和身体动作。

- 参考《无题大鹅》所体现的视觉清晰度、无脸人物、日常场景幽默和克制表达，但不复制其具体角色、建筑、配色、资产或构图。
- 美术必须看起来可以由小型团队在 Blender 和游戏引擎中稳定制作，不能依赖绘画式细节或只有生成图片才能维持的复杂纹理。

### 角色造型

- [ ] 人物使用简单低多边形身体、宽松衣物和清晰剪影。
- [ ] 人物面部完全不绘制五官，不出现眼睛、鼻子、嘴或贴图表情；感情通过站姿、手臂、身体倾斜和动作节奏表达。
- [ ] 鸡保持圆润、偏胖、重心较低的几何造型；不用写实羽毛。
- [ ] 鸡的情绪主要通过歪头、伸脖、压低身体、翅膀开合、步幅和羽毛整体轮廓表达。
- [ ] 猫、麻雀和黄鼠狼都使用少量大块面塑造，第一眼即可从轮廓识别。

### 场景与镜头

- [ ] 主院子采用略微俯视的三分之四视角，接近正交投影，保证道路、领地、虫点和危险动静清楚可读。
- [ ] 2D 横版追鸡仍使用相同低多边形模型、配色和光照，只将镜头转为侧视，不切换成另一套插画风格。
- [ ] 场景结构简洁，保留足够空地让鸡的移动、脚印、花、虫子和灯光成为视觉重点。
- [ ] 植物使用大叶片和成组块面，不制作密集写实树叶、草丝和高频噪点。

### 材质、颜色与光照

- [ ] 材质以无贴图或极简贴图的平涂块面为主，允许轻微色差，但不使用水彩、油画、彩铅、厚涂笔触或照片材质。
- [ ] 基础配色使用奶油白、柔和草绿、灰蓝、砖红和稻草黄；整体低饱和、明亮、温暖。
- [ ] 音乐、后背大灯等荒诞事件才使用少量青色、粉色和紫色，结束后恢复克制色彩。
- [ ] 白天使用柔和日光和清楚但不过黑的投影；夜晚使用深蓝环境色与少量暖黄灯光，确保鸡、虫和黄鼠狼的剪影可读。
- [ ] 雨水、泥点、羽毛、尘土和粪便都做成简洁的大形状特效，不追求写实液体或脏污纹理。

### 道具与特效

- [ ] 每件可互动道具只有一个强轮廓和一个主要动作：水盆要能看到水面，音箱有夸张旋钮，大灯明显沉重，喷水器的水弧清楚可见。
- [ ] 普通可互动物体不依赖持续发光描边；优先用摆放位置、动画、声音和角色朝向提示。
- [ ] 舞池灯光、后背大灯光柱、惊飞羽毛和粪便脚印可以夸张，但粒子数量保持克制。
- [ ] 粪便体积小、色彩低调，不添加苍蝇、臭气或恶心细节；花朵稀疏并保持明显间距。

### 动画原则

- [ ] 动画强调预备动作、停顿和反应，不追求大量连续小动作。
- [ ] 喜剧动作可以夸张：踩粪后双手高举、落汤鸡缩成一团、大灯后坐力震乱羽毛。
- [ ] 温暖动作应放慢：喝水后舒展、凉坑里摊平、被抱住后逐渐安静、结局时主动靠近。
- [ ] 不使用写实人物表演或面部特写，镜头始终相信身体语言。

### UI 风格

- [ ] UI 使用简单圆角块面、低饱和底色和清楚文字，不遮挡主要院子区域。
- [ ] 正式游戏尽量不显示精确身体数值；通过动作和场景反馈表达。精确亲密度等数值只保留在调试面板。
- [ ] 不使用复杂羊皮纸、写实木纹、霓虹边框或与低多边形场景无关的插画装饰。

### 禁止偏离

- 不做写实、像素、二次元或绘本厚涂风格。
- 不给人物补画五官。
- 不堆叠生成式图片常见的密集植物、随机装饰、复杂材质和无意义发光颗粒。
- 不为单个事件制作与主游戏不一致的高精度宣传画式画面。
- 每个最终资产必须能在相同低多边形建模、平涂材质和统一光照规则下被后续资产稳定复现。

## 十三、现有基础，不要重复制作

以下内容当前已有可复用实现，不属于“完全未做”：

- 清晨由人寻找蛋、找到前不能回屋、找到后可继续活动。
- 蛋的多地点隐藏、方向叫声、靠近轮廓和超时环境线索。
- 人鸡近景喂食、摸头、顺背、轻抱及关系阶段基础。
- 鸡的移动、啄食、叫声、刨土、冲刺、扑翅和对应场景教学。
- 多类食物、虫类生成、进食、冲刺能量恢复和基础饱食计算。
- 池塘饮水、天气类型、雨天与泥泞的部分基础。
- 松土区、遮阴棚、低栖木及沙浴、休息、停栖动画基础。
- 猫、麻雀、黄鼠狼实体与部分行为基础。
- 14 天流程、存档、结局检查点、失败重试和结局后自由生活框架。
- 蛋相册、调试天数、调试亲密度、跳阶段和生成事件工具。

这些基础可以迁移到新循环，但凡在上文标为 `[~]` 的内容，都不能因为已有代码就视为新设计已经完成。
